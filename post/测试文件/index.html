这是一篇测试用的文章

如你所见这个网站里除了这一篇文章几乎没有别的东西了.

这是一个想给厂子用的测试网站.

这个网站是以 Github Pages 为基础,以 Hugo 软件生成的静态站点.并不依赖微信公众号,淘宝,抖音等软件服务.这是一个完全自主的网页.如果有需要或条件允许,这个网站可以迁移到国内的 Gitee Pages 或者是以自建服务器的方式运行.

但静态网站总归是一种比较古老的网站形式了,它无法实现一些(相对而言)较先进的功能,比如调用数据库来自动生成站点,或者是在文章下显示评论等等.但也只能如此了.

希望这个网站能够有点作用.
